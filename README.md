# My 2D Unity Game

A 2D platformer game developed in Unity.  
Focuses on clean architecture, reusable scripts, and game mechanics learning.

## Features
- Player movement and animations
- Enemy AI
- Level design with prefabs
- Scoring system

## Tools
- Unity 2023.x
- C# scripts
- Git for version control

## Notes
This project demonstrates Unity fundamentals and clean code practices.
